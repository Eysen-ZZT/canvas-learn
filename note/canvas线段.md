## ctx.lineWidth 属性

`lineWidth` 属性用于设置线段厚度的属性 ( 即线段的宽度 )

当获取属性值时，它可以返回当前的值 ( 默认值是1.0 )

当给属性赋值时， 0、 负数、 Infinity 和 NaN 都会被忽略；除此之外，都会被赋予一个新值

### 语法

```
ctx.lineWidth = value;
```

| 值    | 说明                                                     |
| ----- | -------------------------------------------------------- |
| value | 描述线段宽度的数字。 0、 负数、 Infinity 和 NaN 会被忽略 |
